3 Linear Maps

3.B Null Spaces and Ranges​

NullSpace And Injectivity​

We begin with the set of vectors that get mapped to 0.

3.12 Definition nullsapce, null TTT​

For T∈L(V,W)T\in \mathcal{L}(V, W)T∈L(V,W), the null space of T,

null T={v∈V:Tv=0}\text{null}\ T = \{v\in V : Tv = 0\}null T={v∈V:Tv=0}

3.14 Theorem The null space is a subspace​

For T∈L(V,W)T\in \mathcal{L}(V, W)T∈L(V,W). Then null T\text{null}\ Tnull T is a subspace of VVV.

3.15 Definition Injective​

A function T:V→WT: V \rightarrow WT:V→W is call injective if Tu=TvTu=TvTu=Tv implies u=vu=vu=v.

  • TTT is injective if u≠vu\not ={v}u=v implies Tu≠TvTu\not =TvTu=Tv
  • TTT is injective if it maps distinct inputs to distinct outputs
  • one-to-one equals injective

3.16 Theorem Injectivity is quivalent to nullspace equals {0}\{0\}{0}​

For T∈L(V,W)T\in \mathcal{L}(V, W)T∈L(V,W). Then TTT is injective if and only if null T={0}\text{null}\ T = \{0\}null T={0}.

Thus we can check whether a linear map is injective by checking whether 0 is the only vector that mapped to 0 by 3.16.

Range and Surjectivity​

We give a name to the set of output vectors of a function.

3.17 Definition range​

For T∈L(V,W)T\in \mathcal{L}(V, W)T∈L(V,W), the range of TTT,

range T=Tv:v∈V\text{range}\ T = {Tv:v\in V}range T=Tv:v∈V

For T∈L(V,W)T\in \mathcal{L}(V, W)T∈L(V,W). Then range T\text{range}\ Trange T is a subspace of WWW.

Fundamental Theorem of Linear Maps​

3.22 Theorem Fundamental Theorem of Linear Maps​

Suppose VVV is finite-dimensional and T∈L(V,W)T\in \mathcal{L}(V, W)T∈L(V,W). Then range T\text{range}\ Trange T is finite-dimensional and

dim V=dim null T+dim range T\text{dim}\ V = \text{dim}\ \text{null}\ T + \text{dim}\ \text{range}\ Tdim V=dim null T+dim range T

3.23 A map to a smaller dimensional space is not injective​

Suppose VVV and WWW are finite-dimensional vector spaces such that dim V>dim W\text{dim}\ V > \text{dim}\ Wdim V>dim W. Then no linear map from VVV to WWW is injective.

3.24 A map to a larger dimensional space is not surjective​

Suppose VVV and WWW are finite-dimensional vector spaces such that dim V<dim W\text{dim}\ V < \text{dim}\ Wdim V<dim W. Then no linear map from VVV to WWW is surjective.
