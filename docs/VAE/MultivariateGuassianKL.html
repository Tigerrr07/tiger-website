KL divergence of two Gaussian distributions

For a multivariate Guassian random variable x∼N(μ,Σ)\bold{x} \sim \mathcal{N}(\boldsymbol{\mu}, \Sigma)x∼N(μ,Σ)，the probability density function is

p(x)=1(2π)n/2∣Σ∣1/2exp⁡(−12(x−μ)TΣ−1(x−μ))p(\bold{x}) = \frac{1}{(2\pi)^{n/2} |\Sigma|^{1/2}} \exp(-\frac{1}{2}(\bold{x} - \boldsymbol{\mu})^T \Sigma^{-1} (\bold{x} - \boldsymbol{\mu}))p(x)=(2π)n/2∣Σ∣1/21​exp(−21​(x−μ)TΣ−1(x−μ))
  • μ\boldsymbol{\mu}μ is n-dimensional mean vector
  • Σ\SigmaΣ is n×nn\times nn×n covariance matrix, Σ=E[(x−μ)(x−μ)T]\Sigma = \mathbb{E}[(\bold{x} - \boldsymbol{\mu})(\bold{x} - \boldsymbol{\mu})^T]Σ=E[(x−μ)(x−μ)T]

Two multivariate Guassian distributions p(x)=N(μ1,Σ1)p(\bold{x}) = \mathcal{N}(\boldsymbol{\mu}_1, \Sigma_1)p(x)=N(μ1​,Σ1​) and q(x)=N(μ2,Σ2)q(\bold{x}) = \mathcal{N}(\boldsymbol{\mu}_2, \Sigma_2)q(x)=N(μ2​,Σ2​).
