generative model

  • Discriminative Model: Learn a probability distribution p(y∣x)p(y|\bold{x})p(y∣x)
  • Generative Model: Learn a probability distribution p(x)p(\bold{x})p(x). Sample to generate new data
  • Conditional Generative Model: Learn p(x∣y)p(\bold{x}|y)p(x∣y). Generate new data conditioned on input labels
Example

x\bold{x}x could be an image, yyy is the image's class.

Notation

Probability density functions (PDFs) and probability mass functions (PMFs), also simply called probability distributions.

Taxonomy of Generative Models​

Variational Autoencoders​

Let x\bold{x}x be a random sample from an unknown distribution p(x)p(\bold{x})p(x). We also call x\bold{x}x ​ an observed variable. We assume that there exists an unobserved (latent) variable z\bold{z}z that generates x\bold{x}x.

How to generate new data​

After training, sample new data like this：

  • Sample z(i)\bold{z}^{(i)}z(i) from prior pθ(z)p_{\boldsymbol{\theta}}(\bold{z})pθ​(z). e.g. Gaussian unconditoned on θ\boldsymbol{\theta}θ, pθ(z)=N(z;0,I)p_{\boldsymbol{\theta}}(\bold{z}) = \mathcal{N}(\bold{z};\bold{0}, \bold{I})pθ​(z)=N(z;0,I)
  • Sample x(i)\bold{x}^{(i)}x(i) from conditional p(x∣z(i))p_{}(\bold{x}|\bold{z}^{(i)})p​(x∣z(i))

How to train the model​

Basic idea 1: maximize the likelihood of data

If we can observe z\bold{z}z for each x\bold{x}x during training, we can train this model by maxmizing the likelihood of each x\bold{x}x conditioned on its observed z\bold{z}z. In other words, we can train a conditional generative model p(x∣z)p_{}(\bold{x}|\bold{z})p​(x∣z).

But we can't observe z\bold{z}z, we need to maximize the marginal likelihood:
